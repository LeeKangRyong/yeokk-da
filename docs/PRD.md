# PRD: 엮다 (Yeokk-da) - Cinematic Memory Weaving

## 서비스 정의

**엮다**는 파편화된 사진과 음악을 AI와의 대화를 통해 한 편의 영화 같은 서사로 엮어내는 **인터랙티브 추억 복원 플랫폼**입니다.

사용자가 던진 추억의 조각(이미지/텍스트)을 AI가 게임처럼 인터뷰하여:
- **게임화된 인터뷰**: 추억을 복원하는 듯한 다이얼로그 기반 입력 (Gamified Interview)
- **음악 주파수 맞추기**: 감성 분석 기반 Spotify BGM 큐레이션 및 라디오 다이얼식 탐색
- **시네마틱 스토리텔링**: 스크롤에 반응하는 한 편의 영화 같은 연출 (Scroll-driven Storytelling)
- **멀티 채널 공유**: 인스타그램 스토리, 카카오톡 등 SNS 최적화 공유 기능

**URL**: https://yeokk-da.netlify.app

---

## 핵심 가치

- **재해석**: 단순 저장이 아닌 AI와 사용자의 대화를 통한 추억의 서사화
- **몰입**: 일반 SNS의 피드 형식을 벗어난 풀스크린 시네마틱 경험
- **교감**: 추억을 복원하는 과정(게임적 인터뷰) 자체에서 오는 감성적 만족
- **확산**: 나만의 시네마틱 페이지를 SNS에 최적화된 형태로 공유 및 자랑

---

## 사용자 플로우

### 추억 생성 & 공유 (Memory Weaving & Share Flow)

```
조각 던지기 → AI 인터뷰 → 음악 주파수 맞추기 → 시네마틱 페이지 생성 → SNS 공유
```

1. **조각 던지기 (Input)**
   - 이미지 업로드 (Azure Blob Storage 연동)
   - 기억의 파편(짧은 메모) 입력

2. **AI 인터랙티브 인터뷰 (Gamified Process)**
   - Claude AI가 업로드된 사진을 분석하여 감성 질문 생성
   - 대화창 형식을 통해 추억 구체화 및 실시간 테마 배경 적용

3. **음악 주파수 맞추기 (Spotify Integration)**
   - 분석된 무드에 어울리는 Spotify 곡 추천 및 탐색 게임화
   - 최종 BGM 확정 및 애니메이션 싱크

4. **시네마틱 페이지 생성 & 저장**
   - AI 서사와 레이아웃이 결합된 결과물 생성

5. **공유 (Share)**
   - **인스타그램 스토리**: 시네마틱 카드 이미지 생성 및 공유
   - **링크 공유**: 카카오톡/트위터 등 고유 URL 전송 (OG Tag 최적화)

## 핵심 기능

### 1. 데이터 소스 및 입력
- **직접 업로드**: 사진/영상 (최대 10장)

### 2. 게임화된 AI 분석 (Claude AI)
- **Interactive Interviewer**: 질문-답변 기반 서사 빌드업
- **Spotify API**: 감정 기반 곡 추천 및 미리듣기 지원
- **Story Generation**: 영화 예고편 스타일의 독백 서사 작성
- **Mood Analysis**: 5가지 애니메이션 테마 매칭

### 3. 시네마틱 추억 페이지 (Framer Motion)
- **레이아웃 5종**: Magazine, Cinema, Parallax, Collage, Music Story
- **Scroll-driven Effect**: 스크롤에 반응하는 시네마틱 연출

### 4. 소셜 공유 기능 (SNS Socializing)
- **Instagram Story Share**: 현재 페이지를 스토리 규격에 맞는 이미지/영상 카드로 변환
- **Dynamic OG Tag**: 공유 링크 접속 시 AI가 쓴 제목과 대표 이미지가 미리보기에 표시
- **Share Token**: 비회원도 접근 가능한 보안 공유 링크(`s/:token`) 생성
